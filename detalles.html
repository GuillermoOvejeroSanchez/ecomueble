Detalles de Ecomueble

Introducción

Esta aplicación web consiste en una página dedicada al intercambio de muebles, con la intención de darle un nuevo uso y así promover el reciclaje de artículos en buen estado. Permite a los usuarios deshacerse de los productos, evitando desecharlos y ayudando a otros usuarios que buscan ese mismo artículo.

El uso es sencillo, un usuario registrado puede subir los artículos que desea desechar y otro usuario registrado contacta con él para obtener el producto.

Para el cambio, implementaremos un sistema de puntos. Cada usuario gana al registrarse 1000 puntos para obtener artículos. Los puntos de cada artículo los determinará el usuario dependiendo del tipo y calidad del producto.

La novedad que presenta esta plataforma es que su uso es íntegramente gratuito, es decir, no pagas por obtener un mueble. Para regular el tráfico de los productos, implementamos una moneda virtual de puntos evitando que un usuario se aproveche de los recursos.

Usuarios

  • No registrado:

    Usuario que entra en la pagina por primera vez o buscando productos que puedan interesarle para mas tarde registrarse y comprarlos

    • Registrarse
    • Iniciar sesión
    • Ver y buscar productos
    • Acceder a las páginas de información de la web (TyC)
    • Ponerse en contacto con los administradores mediante el formulario web
  • Registrado:

    Usuarios que han decidido comprar o vender algun mueble en la plataforma de Ecomueble y estan familiarizados con la pagina

    • Funcionalidades del usuario no registrado
    • Publicar/Borrar productos propios
    • Comprar productos
    • Contactor con otros usuarios mediante el chat
    • Ver perfiles de otros usuarios
    • Ver y editar su pagina de perfil
    • Reportar usuario o producto
  • Administrador:

    El administrador se encarga del mantenimiento de la web y de gestionar todo el contenido, ademas de poder moderar el contenido en caso de que se incumplan las condiciones

    • Funcionalidades anteriores de los otros tipos de usuario
    • Editar y borrar productos de otros usuarios
    • Eliminar cuentas de usuarios
    • Bloquear el acceso temporal o permanentemente de usuarios
    • Revisar chats entre usuarios con objetivo de moderacion
